## 이 시리즈에서 만드는 것

Python과 FastAPI로 **HTTP API 서버**를 처음부터 만든다.

```
[앱 / 웹브라우저]
      │  GET /users       "유저 목록 줘"
      │  POST /users      "새 유저 만들어줘"
      │  DELETE /users/1  "1번 유저 지워줘"
      ▼
[FastAPI 서버]  ← 이걸 만든다
      │
      ▼
[데이터베이스]
```

시리즈를 마치면 유저 데이터를 조회·생성·삭제하는 API 서버를 직접 만들 수 있다.
